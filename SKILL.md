# Code Insight Skill

Interactive code visualization tool that builds focused UML class diagrams
(**map mode**) and numbered call-flow traces (**trace mode**) for a target
repo, rendered in a local React Flow viewer at [`web/`](./web).

**All operating rules live in [`AGENTS.md`](./AGENTS.md) — read it first.**
